What you're watching
FHP Frame Handshake Protocol — two AI models agree on rules before doing any work together. Like establishing ground rules before a meeting.
SSV Shared State Vector — a live dashboard both models read to check if the session is still on track. Drift detected here triggers a warning before it becomes a problem.
LOCK The moment both models commit to the agreed rules. Nothing changes without an explicit renegotiation — no silent drift allowed.
ADAPT The only legitimate way to change the rules mid-session. Must be proposed, acknowledged, and committed — never unilateral.
Human Router The human operator who sequences messages between models and holds final authority. All escalations surface here first.
drift_delta A score measuring how far the session has drifted from the agreed frame. Watch the chart — it spikes during the stress test, then recovers.
